# EIP Generator

## What is it ?
EIP generator is a Yeoman generator to kickstart Ethereum Improvement Proposal.

## How to use it ?

### Install Prerequisites

- [npm](https://www.npmjs.com/) package manager
- [yeoman](https://yeoman.io/)

#### Install Yeoman
```shell script
npm install -g yo
```
